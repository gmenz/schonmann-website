# schonmann-website

Personal academic website for Roberto H. Schonmann, Emeritus Professor in the
Department of Mathematics at UCLA.

Plain static HTML + CSS, no build step. Served via GitHub Pages.

## Files

- `index.html` — Home / bio
- `publications.html` — Publication list (probability, cooperation, economics, QM)
- `teaching.html` — Teaching history and doctoral students
- `cv.html` — Structured CV summary; links to UCLA PDF
- `collaborators.html` — Co-authors by area
- `style.css` — Minimal academic styling

Content drawn from the existing UCLA faculty page at
`https://www.math.ucla.edu/~rhs/` as of May 2026.
